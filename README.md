**Technology stack:**
+ Kotlin
+ Jetpack
  + Compose for UI and navigation
+ Retrofit for networking
+ Kotlin Coroutines for asynchronous work
+ Coil for loading images asynchronously
+ Architecture
  + MVVM
  + Repository pattern
